1. 다음 set을 비트로 표현하시오.
{ 0, 1, 2, 3, 4 } => 11111
{ 1, 2, 3, 4 } => 11110
{ 1, 2, 4 } => 10110
{ 2, 4 } => 10100
{ 1 } => 00010

2. 1번 값을 십진수로 바꾸시오. 
   31, 30, 22, 20, 2
4. 1010에서 i번째 비트의 값을 1로 변경하려고 한다. i = 2 이라고 가정할 때, 결과와 방법을 쓰시오.  
  (1) 결과  1110
  (2) 필요 연산  1010 | 1 << 2
4. 1010 의 세번째 비트 값을 알기 위한 연산식을 쓰시오
  1010  & 1 << 3
